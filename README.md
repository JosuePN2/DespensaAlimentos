# Minha Despensa

## Descrição

Um aplicativo móvel para gestão inteligente de itens domésticos, focado no controle de datas de validade para evitar desperdício.

## Funcionalidades

- Gestão de itens: adicionar, listar e remover itens em tempo real.
- Receber notificações quando um alimento estiver perto de vencer
- Ordenar alimentos por ordem de vencimento ou ordem alfabética
- Alertas visuais que indicam a proximidade do vencimento

## Tecnologias

- Frontend: React Native com Expo
- Linguagem: Typescript
- Supabase
